# 💰 Personal Finance Tracker

A full-stack Personal Finance Tracker built using the MERN Stack. It helps users manage their income and expenses, track spending, and view financial insights through an interactive dashboard.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- ➕ Add Transactions
- ✏️ Edit Transactions
- 🗑️ Delete Transactions
- 📊 Dashboard with Balance, Income & Expense
- 🥧 Expense Chart
- 🔍 Search Transactions
- 📂 Filter by Transaction Type
- ↕️ Sort Transactions
- 💵 Budget Tracker
- 📈 Statistics
- 📑 Category Summary
- 📄 Export Transactions to PDF
- 📊 Export Transactions to CSV
- 📱 Responsive Design

---

## 🛠️ Tech Stack

### Frontend
- React.js

- Bootstrap

- react-csv
- jsPDF

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/personal-finance-app.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
